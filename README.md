GithubTrending
==============

An unofficial API for retrieving Github's trending repositories written in Node.js.

Properties:
- Repo full name
- Repo owner
- Repo description
- Repo URL
- Repo's starsToday

<h2> How to Use </h2>
Simply add the desired languages as 'languages' parameters to your url and get a an array of trending repos from all the languages in one JSON object.

<b> Example: localhost:5000/trending?languages=objective-c&languages=ruby&languages=trending </b>
